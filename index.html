Architecture

Roll-up OS｜3層アーキテクチャ

Roll-up OS | Three-Layer Architecture

Layer 3 — Intelligence
AIエージェント & LLM
AI Agents & LLM

自律型AIエージェントによる業務自動化と意思決定支援。不動産仲介、在庫管理、顧客対応など、産業特化型SKILLSパッケージとして複数の産業に横展開する。

Autonomous AI agents for operational automation and decision support. Delivered as industry-specific SKILLS packages deployable across multiple verticals.

LLM Autonomous Agent SKILLS Package
Layer 2 — Data
データ統合基盤
Unified Data Platform

各産業のERP・SaaS・IoTから一次データを収集・統合する中間層。秘密計算によるプライバシー保護、メタデータ管理、RAGパイプラインを備え、AIレイヤーに構造化されたデータを供給する。

The middleware layer that collects and unifies first-party data from ERP, SaaS, and IoT across industries. Equipped with secure computation, metadata management, and RAG pipelines to feed structured data to the AI layer above.

Data Connector MPC Metadata RAG
Layer 1 — Trust
ブロックチェーン信頼レイヤー
Blockchain Trust Layer

AIの行動ログをブロックチェーンに記録し、透明性と説明責任を担保する信頼基盤。資産のトークン化・所有権移転の改ざん耐性と監査証跡を提供し、UniCaskやANGOのNFTもこのレイヤーで動く。

Records AI action logs on blockchain to ensure transparency and accountability. Provides tamper-resistant audit trails for asset tokenization and ownership transfer. UniCask and ANGO NFTs run on this layer.

Tamper-proof Log Audit Trail Provenance Tokenization
